Good First Issues

This is a list of JavaScript repositories with good first issues for newcomers to open source. Contributions are welcome!

This list gets updated every day at midnight.

brave/brave-browser

  • Version number in "About Brave" should be orange to indicate that it is a link
  • Consider changing return type of NTPBackgroundPrefs::GetSelectedValue() to std::string

cuttle-cards/cuttle

  • [Bug]: GameOverDialog logo images are too large
  • [Feature]: Stats Page - Enable creating a link to a specific season's leaderboard
  • [DevEx]: Update PR template to expect an issue link

WordPress/gutenberg

  • File Block: file name needs to use PlainText component

  • Documentation: spacing.blockGap explanation is inadequate for custom blocks
  • Create Block: Document how to work with the templates defined with render
  • Consistent terminology: Patterns vs. Block Patterns
  • Navigation block shows unintuitive message if no menus and no pages exist
  • Avatar block not 1:1 between the editor and the front-end
  • @wordpress/scripts doesn't detect render PHP files without a restart
  • DEWP docs and implementation mismatch/confusion in regards of assets file name
